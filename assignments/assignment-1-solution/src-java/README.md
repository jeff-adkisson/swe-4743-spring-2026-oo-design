# Crazy Eights (Java)

Java port of the Crazy Eights C# solution with the same gameplay and structure.

> This Java solution was AI-generated by Codex in VS Code from the C# solution. The operation should match the C# solution and the structure should match the requirements document.

## Layout

- Source: `src-java/src/main/java/edu/kennesaw/crazy8s`
- Entry point: `edu.kennesaw.crazy8s.Main`

## Run (CLI)

From `assignment-1/solution./src-java`:

```bash
javac -d out $(find src/main/java -name "*.java")
java -cp out edu.kennesaw.crazy8s.Main [randomSeed] [handSize] [showAllHands]
```

Arguments (optional):
- `randomSeed`: Integer used to seed the random number generator for reproducible games, for example `42`. Default `0`
- `handSize`: Integer for starting cards per player, for example `5`. Default `5`
- `showAllHands`: Boolean-like flag to reveal all hands, use `true` to show or `false` to hide (example `true`). Default `false`

## Run (Docker)

From `assignment-1/solution./src-java`:

```bash
docker build -t crazy8s-java .
docker run -it --rm crazy8s-java [randomSeed] [handSize] [showAllHands]
```
